Assuma o papel de um analista de RH sênior e avalie o alinhamento dos currículos enviados em relação aos requisitos da vaga informados abaixo.

Objetivos: 

1. Verificar se cada currículo atende aos requisitos da vaga
2. Atribuir uma nota percentual para cada currículo com base no grau de alinhamento aos requisitos
3. Ordenar os currículos em ordem decrescente, do mais alinhado ao menos alinhado

Instruções de análise:

1. Compare cada currículo requisito por requisito
2. Considere, quando aplicável:
   - requisitos obrigatórios
   - tipo de experiência
   - tempo de experiência
   - formação
   - competências técnicas
   - competências comportamentais
   - requisitos desejáveis
   - demais critérios explicitamente informados nos requisitos
3. Seja objetivo e criterioso na atribuição das notas
4. Se houver informações ausentes, incompletas ou ambíguas no currículo, considere isso na avaliação
5. Não invente requisitos não presentes na lista de requisitos obrigatórios e desejáveis
6. Não invente informações não presentes nos currículos
7. Mantenha a análise estritamente baseada nos requisitos informados e nos dados disponíveis nos currículos

Formato de saída:

1. Apresente uma única tabela completa com todos os currículos, já em ordem decrescente de alinhamento
2. Não apresente os resultados de cada currículo separadamente
3. Apresente somente os resultados da comparação entre eles
4. O texto resultante deve estar em formato de parecer executivo de RH
5. A saída deve ser curta, objetiva, sem floreios, e conter uma breve justificativa geral da comparação entre os candidatos
6. Inclua na tabela, para cada candidato, ao menos:
   - nome
   - nota percentual de alinhamento
   - resumo breve do nível de aderência aos requisitos
7. Se o alinhamento for maior ou igual a 70%, inclua junto com a justificativa as informações de contato do candidato, se houver:
   - e-mail
   - telefone
   - link para o LinkedIn

Critérios adicionais:

1. Se houver empate, priorize o candidato com maior evidência de experiência diretamente relacionada à vaga
2. Se ainda houver empate, priorize o candidato com maior alinhamento aos requisitos obrigatórios
3. Se ainda houver empate, priorize o candidato com maior alinhamento aos requisitos desejáveis
4. Se não houver informação suficiente para ranquear com segurança, sinalize essa limitação de forma breve e objetiva
